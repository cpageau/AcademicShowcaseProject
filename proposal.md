## Project Overview

This project will investigate how we can optimally place railway stations within the existing transportation network so that all areas in the continental United States are within a specified distance of a railway station. Using the METAL graph data, the goal is to minimize the number of additional stations needed to cover all the remaining regions. 
## Problem Statement

Transportation infrastructure especially public infrastructure has been a topic of much debate among much Americans. Many people advocate for more public transport while many believe that such a undertaking is too costly and the country is too vast and wide for a country-spanning public infrastructure. Railways has been much of a conversation in this matter. Therefore, this project will help us to answer how much of the United States is already connected and a rough approximation how much more infrastructure we need to much all of the United States connected with rail.
